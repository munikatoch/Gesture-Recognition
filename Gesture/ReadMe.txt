# Projects
Mouse Gesture Recognition
This project will execute a default gesture or a user defined profram or file when some one drag the mouse down, up, left, right on the
provided frame (window).
Default gesture :- 1. Up - IPConfig command result (IP, MAC Address, etc of all adapters)
                   2. Down - MAC address of all adapters
                   3. Right - System restart
                   4. Left - System shutdown
